    BulkAll is a generic helper that will partition any lazy stream of documents and send them to OpenSearch as concurrent bulk requests

    The index to target will be inferred from T. If no default index has been mapped for T using OpenSearch.Client.ConnectionSettingsBase`1.DefaultMappingFor``1(System.Func{OpenSearch.Client.ClrTypeMappingDescriptor{``0},OpenSearch.Client.IClrTypeMapping{``0}}) on ConnectionSettings, an exception will be thrown. Inference can be overridden using Index, and in addition, an index can be specified for each document using BufferToBulk.
